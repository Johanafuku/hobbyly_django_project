# hobbyly_project
Django project to create a web like instagram to talk about hobbies and favorite activities with others.

## Features
- Register as user.
- Login.
- Create posts with images and text. 
- Comment on posts.
- Like, dislike posts.
- Erase comments.
- Have your own profile.
- Contact with the web.
